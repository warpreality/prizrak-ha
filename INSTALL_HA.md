# Установка интеграции в Home Assistant

## Способ 1: Ручная установка (для тестирования)

1. **Скопируйте интеграцию в Home Assistant:**
   ```bash
   # На сервере Home Assistant
   cp -r custom_components/prizrak /config/custom_components/
   ```

2. **Перезапустите Home Assistant:**
   - Через UI: Settings → System → Restart
   - Или через командную строку: `ha core restart`

3. **Добавьте интеграцию:**
   - Settings → Devices & Services
   - Нажмите "Add Integration"
   - Найдите "Призрак"
   - Введите учетные данные

## Способ 2: Через HACS (после публикации)

1. Установите HACS (если еще не установлен)
2. Добавьте этот репозиторий как Custom repository
3. Установите интеграцию через HACS
4. Перезапустите Home Assistant
5. Добавьте интеграцию через UI

## Проверка установки

После установки проверьте логи Home Assistant:

```bash
# Просмотр логов
ha core logs | grep prizrak
```

Или через UI: Settings → System → Logs

## Отладка

Если интеграция не работает:

1. **Проверьте логи:**
   - Включите debug логирование для интеграции:
   ```yaml
   logger:
     default: info
     logs:
       custom_components.prizrak: debug
   ```

2. **Проверьте конфигурацию:**
   - Убедитесь, что учетные данные правильные
   - Проверьте, что device_id существует

3. **Проверьте зависимости:**
   - Убедитесь, что установлены все зависимости из `manifest.json`

## Известные проблемы

- Если SignalR не подключается, проверьте доступность `wss://monitoring.tecel.ru`
- Если авторизация не работает, проверьте формат ответа API (может потребоваться корректировка парсинга)
